void HAL_GPIO_EXTI_Callback ( uint16_t GPIO_Pin)

GPIO EXTI callback.

This function is called by the HAL when an external interrupt occurs on a configured GPIO pin and transmit a character through UART2 interface (SIMULATION only).

Parameters
[in]GPIO_PinSpecifies the pins connected to EXTI line.
Note
This function is a user implementation of the weak HAL_GPIO_EXTI_Callback() provided by the STM32 HAL.

void HAL_TIM_PeriodElapsedCallback ( TIM_HandleTypeDef * htim)

Period elapsed callback in non blocking mode.

Note
This function is called when TIM9 interrupt took place, inside HAL_TIM_IRQHandler(). It makes a direct call to HAL_IncTick() to increment a global variable "uwTick" used as application time base.
