Detailed Description

Battery monitoring module.

Author
A. Bellina

This module provides APIs to measure battery level using the ADC and to evaluate thresholds for battery alarms. It is meant to be used inside a FreeRTOS task but can also be called from bare-metal code.

Features:

  • ADC acquisition wrapper
  • Configurable voltage thresholds
  • Integration with GPIO alarm LED

Dependencies:

  • HAL ADC driver
  • HAL GPIO driver
